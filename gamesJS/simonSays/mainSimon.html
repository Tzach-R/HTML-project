Rules of the Game:

  1. Click the Power button to start the program.
  2. Click the Start button to start the game itself.
  3. Wait and pay attention as Simon gives the first signal. Repeat the signal by pressing the same color.
  4. Simon will get the first input and add one. That process will repeat for twenty times.
  5. If you make a mistake, don't worry! Simon will tell you but your counter will stay at the same level.
  6. You win when you get everything correctly and Simon will light up with joy!

Strict Mode:

  1. Click the Strict button to activate strict moode.
  2. In this mode if you make a mistake your counter will be reset and you'll have to start the game all over again.
